BIKING PEOPLE WORLD — SITE PACKAGE

Everything in one upload. 109 pages, all validated, no merge-conflict markers.

--------------------------------------------------------------------
HOW TO UPLOAD (important)
--------------------------------------------------------------------
GitHub sometimes MERGES a same-named file instead of replacing it,
which produced the duplicated Gxygle page (the stray "=======" line).
To avoid it: DELETE the folder on GitHub first, then upload fresh.
Always include .nojekyll and the assets/ folder.
