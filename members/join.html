Get into Biking People World

Follow these steps to connect. Make sure you've installed the modpack first.

Server Address

bikingpeople.exaroton.me

How to Connect

  1. Install the Biking People World modpack (NeoForge 1.21.1).
  2. Open Minecraft with the modpack loaded.
  3. Go to Multiplayer → Add Server, and paste the address above.
  4. Join! If the server looks offline, it may be asleep — it wakes up shortly after someone connects (exaroton servers sleep when empty).

World Rules

  • 1Be respectful to other players. No griefing or stealing.
  • 2Don't destroy or alter other players' builds without permission.
  • 3Keep things friendly — this is a community world.
  • 4Have fun, build cool things, and explore the cities, MiniGame Land, and beyond.

Rules are easy to edit — change this list anytime to match how you run the world.
